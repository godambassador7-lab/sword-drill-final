# Paleo-Hebrew Pronunciation and Course Repository (v3)

This repository provides:

- Detailed Paleo-Hebrew letter data with reconstructed theoretical pronunciation.
- A mega-pack of 300+ roots explained in a Paleo-style semantic framework.
- SVG artwork for each Paleo letter with a transparent-style background.
- A full three-level course module suitable for integration into an app such as Sword Drill.
