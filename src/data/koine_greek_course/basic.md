# BASIC KOINE GREEK

## 1. Alphabet
- Α α — alpha — “ah”
- Β β — beta — “b”
- Γ γ — gamma — “g” (before κ γ χ ξ = nasal “ng”)
- Δ δ — delta — “d”
- Ε ε — epsilon — “eh”
- Ζ ζ — zeta — “dz”
- Η η — eta — “ay”
- Θ θ — theta — “th” (as in thick)
- Ι ι — iota — “ee”/“ih”
- Κ κ — kappa — “k”
- Λ λ — lambda — “l”
- Μ μ — mu — “m”
- Ν ν — nu — “n”
- Ξ ξ — xi — “ks”
- Ο ο — omicron — “o” (short)
- Π π — pi — “p”
- Ρ ρ — rho — “r” (slightly rolled)
- Σ σ/ς — sigma — “s”
- Τ τ — tau — “t”
- Υ υ — upsilon — “ü”
- Φ φ — phi — “ph”
- Χ χ — chi — “kh”
- Ψ ψ — psi — “ps”
- Ω ω — omega — “ō”

## 2. Syllables & Breathing Marks
- Smooth (᾿) = no “h”
- Rough (῾) = initial “h”
Example: ἁγιος — hagios
