# INTERMEDIATE KOINE GREEK

## 1. Noun Cases
- Nominative — subject
- Genitive — of/from
- Dative — to/for/in
- Accusative — direct object
- Vocative — calling

## 2. Verb System Overview
Tenses: Present, Imperfect, Future, Aorist, Perfect, Pluperfect  
Voices: Active, Middle, Passive  
Moods: Indicative, Subjunctive, Imperative, Infinitive, Participle
